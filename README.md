# Env
Python package manager. Inspired by yarn.

## Usage

1. Generate `package.json` file and `.env` directory.

```bash
env init
```

2. Install package, which would generate `python_modules` directory.

```bash
env install <package>
```

3. Install package.

```bash
env uninstall <package>
```

4. Show all packages.

```bash
env show
```
